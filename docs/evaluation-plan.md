# Evaluation Plan

This evaluation plan is designed for the **prototype validation stage / 原型验证阶段**. It focuses on whether the workflow can improve Chinese content texture without changing the original intent.

## Test Materials

- WeChat public account article drafts
- Xiaohongshu copy drafts
- Short video spoken scripts
- Novel prose excerpts
- Marketing copy drafts
- Long-form drafts with multiple sections

## Evaluation Metrics

### Meaning Preservation

Check whether the rewritten text keeps the original argument, facts, emotional direction, and key details.

### AI-Flavor Reduction

Evaluate whether the output reduces template-like transitions, repetitive structures, vague summaries, and over-explaining.

### Platform Style Adaptation

Check whether the text fits the target platform's expected rhythm, density, tone, and reader context.

### Naturalness

Review whether the output sounds like a fluent human-edited Chinese draft rather than a generic generated response.

### Factual Consistency

Verify that the rewrite does not introduce unsupported claims, invented examples, fake data, or unverified conclusions.

### Long-Form Consistency

For longer drafts, check whether terminology, viewpoint, tone, and narrative direction remain stable across sections.

### Output Stability

Run similar inputs through the workflow and compare whether diagnosis categories and rewrite quality remain reasonably consistent.

## Review Method

Each sample can be reviewed with a simple score from 1 to 5 for each metric, plus short reviewer notes. Low scores should trigger another pass through the Diagnosis Agent and Self-Check Agent.
